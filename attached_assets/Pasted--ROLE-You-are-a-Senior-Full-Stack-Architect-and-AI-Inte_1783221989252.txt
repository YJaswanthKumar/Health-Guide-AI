# AGENT INPUT CONTRACT

The backend constructs the input.

Never allow frontend to manually build CrewAI JSON.

Example

Input

{
    "user_profile": {},
    "medical_conditions": [],
    "current_medications": [],
    "daily_logs": [],
    "current_tasks": [],
    "health_assessment": {},
    "nutrition_plan": {},
    "calendar": {},
    "conversation_history": []
}

The backend gathers these from PostgreSQL before invoking an agent.

---

# AGENT OUTPUT CONTRACT

Every CrewAI deployment returns structured JSON.

Never parse natural language.

Consume JSON only.

Example

{
    "backend_actions": [],
    "today_tasks": [],
    "notifications": [],
    "dashboard_companion": {},
    "care_plan": {},
    "warnings": [],
    "recommendations": []
}

Backend interprets the JSON.

Frontend only renders processed data.
